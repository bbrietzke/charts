Welcome to Charts

This is a collection of helm charts that I've worked on for my Kubernetes cluster at home.

Chart Layout

  • volumes: creates a number of persistent volumes to be used by anybody in the cluster

Automation

There is a Makefile included that will help automate a number of the steps to publish the charts.

Commands include:

  • publish: use MkDocs to publish to gh-pages branch
  • volumes: package the volumes chart and move it to the correct location
  • index: create or update the index.yaml file
